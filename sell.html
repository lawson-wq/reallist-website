Viewing request model

No default open homes does not mean no viewings. It means viewings are created from real buyer requests, not empty calendar habits.

Every genuine viewing request is followed up, arranged where access is available, and recorded for the seller.

If the data shows broader demand, an open home can still be arranged. The point is not to avoid viewings. The point is to avoid theatre and track real intent.
